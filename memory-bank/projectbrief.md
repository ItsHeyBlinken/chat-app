# Project Brief

## Goal
Ship a mobile-first, text-only, real-time chat MVP that works in a phone browser, supports anonymous access, and uses an 18+ self-attestation gate.

## MVP Scope
- One global chat room
- Auto-generated guest ID stored locally
- Realtime messaging via Socket.IO
- Persist and bootstrap the last ~20–50 messages from Postgres
- Basic rate limiting (anti-spam)

## Non-Goals (for MVP)
- Accounts, profiles, moderation dashboards, reports, multi-room, images/voice/video
