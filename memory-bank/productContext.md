# Product Context

## What this is
An anonymous, 18+ self-attested, mobile-first global chat with fast realtime messaging.

## Who it's for (initially)
Unknown yet. The MVP is intentionally neutral so it can evolve into a consumer app, a community SaaS product, or an embeddable widget later.

## Success criteria (MVP)
- Works well in a mobile browser
- Users can join without login
- Messages appear for everyone with minimal delay (target: <1–2 seconds)
- Stable connection handling and basic anti-spam guardrails
